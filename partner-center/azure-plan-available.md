# <a name="azure-services-available-in-the-azure-cloud-solution-provider-csp-program"></a>Služby Azure dostupné v programu Azure Cloud Solution Provider (CSP)

**Příslušné role**

- Agent správce
- Správce fakturace
- Globální správce
- Agent helpdesku
- Agent prodeje
- Správce správy uživatelů

## <a name="available-azure-services-in-azure-csp"></a>Dostupné služby Azure v rámci programu Azure CSP

Tento článek obsahuje seznam služeb Azure, které nejsou v programu Azure Cloud Solution Provider (CSP) k dispozici. Také popisuje dostupnost služby v národních cloudech [Microsoft Azure (Německo)](https://azure.microsoft.com/overview/clouds/germany/) a [Microsoft Azure Government](https://azure.microsoft.com/overview/clouds/government/).

>[!Note]
> [Azure Čína](https://www.azure.cn/) není v programu Azure CSP k dispozici.

## <a name="global-cloud"></a>Globální Cloud

Všechny služby založené na modelu Azure Resource Manager jsou k dispozici v programu CSP.  Jiné než Azure Resource Manager služby nejsou v programu CSP k dispozici.  

## <a name="csp-specific-service-configurations"></a>Konfigurace služby CSP-Specific

Následující služby vyžadují speciální konfigurace v CSP:

- [StorSimple](/azure/storsimple/storsimple-partner-csp-overview)

- [Azure Active Directory Domain Services](/azure/active-directory-domain-services/active-directory-ds-csp)

- [Key Vault](https://azurecsp.blob.core.windows.net/files/key-vault-in-csp.docx)

- [Azure Time Series Insights](https://azure.microsoft.com/services/time-series-insights/) Pouze uživatelé z tenanta zákazníka mají přístup k datům v jejich Time Series Insights prostředí. Partneři můžou ve výchozím nastavení spravovat své Time Series Insights prostředí zákazníka, ale pokud potřebují přístup k datům, musí se do tenanta zákazníka přidat.

## <a name="visual-studio-marketplace"></a>Visual Studio Marketplace

Nyní můžete z Visual Studio Marketplace koupit níže uvedené položky, s výjimkou rozšíření třetích stran.

- [Azure DevOps](https://www.visualstudio.com/team-services/)

- [Předplatná sady Visual Studio](https://www.visualstudio.com/subscriptions/)

- [Školení Xamarin University](https://marketplace.visualstudio.com/items?itemName=ms.xamarin-university)

Abychom vám pomohli začít, vytvořili jsme videa a dokumentaci o [tom, jak nastavit, koupit a spravovat Azure DevOps](/vsts/billing/csp/set-up-csp-customer) ve zprostředkovateli CSP.

## <a name="azure-marketplace-items-in-azure-csp"></a>Položky Azure Marketplace v rámci programu Azure CSP

V předplatných Azure CSP nejsou aktuálně k dispozici všechny Azure Marketplace položky.

- Služby Azure založené na Microsoft: tyto služby jsou k dispozici. Přečtěte si předchozí tabulku a komentáře.

- Přineste si vlastní licenci (BYOL): tyto položky jsou k dispozici. Úplný seznam položek Azure Marketplace s podporou BYOL je k dispozici na [stránce Azure Marketplace BYOL](https://azuremarketplace.microsoft.com/marketplace/apps?filters=byol).

- Průběžné platby dle aktuálního využití Azure Marketplacech položek třetích stran: tyto položky jsou k dispozici, pokud byl poskytovatel publikován na kanál CSP. Další informace najdete v tématu [prodej předplatných pro Azure Marketplace Products](csp-commercial-marketplace-overview.md).

- Citrix XenApp Essentials: partneři můžou koupit XenApp Essentials pro zákazníky v CSP. Další informace najdete v následujícím blogu společnosti Citrix [distribuce XenApp Essentials nyní k dispozici prostřednictvím kanálu poskytovatele řešení Microsoft Cloud](https://www.citrix.com/blogs/2018/02/01/xenapp-essentials-now-available-through-microsoft-cloud-solution-provider-channel/).
