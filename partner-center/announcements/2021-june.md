## <a name="perpetual-software-price-list-corrected-and-republished"></a><a name="4"></a>Oprava a opětovné publikování ceníku časově neomezeného softwaru 

### <a name="categories"></a>Kategorie

- Datum: 4. 6. 2021
- Nabídky/trhy

### <a name="summary"></a>Souhrn

Ceník časově neomezeného softwaru publikovaný 1. června 2021 byl 3. června znovu publikován, aby se opravil několik problémů. 

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Nepřímí poskytovatelé a partneři s přímým vyúčtováním, kteří v Cloud Solution Provider provádí transakce s Cloud Solution Provider softwarem.  

### <a name="details"></a>Podrobnosti

1. Nesprávné ceny v některých měnách

   - U některých produktů v ruštině (PRODUCTS) a South Korean Won (KRW) byly od 27. dubna 2021 do 3. června 2021 nesprávné ceny. Tyto produkty měly nesprávné ceny, které byly opraveny v aktuálních trvalých ceníkech.
 
     - Produktová rodina Windows Serveru
     - Produkty SQL
     - Produkty Office
 
   - Problémy s cenami se zjistily dostatečně včas, aby se zajistilo, že partnerům se budou fakturovat správné částky, pokud byly za jakékoli nákupy přeúčtovány, protože se odeslaly nesprávné ceny. Ovlivnění partneři uvidí na své faktuře kredit na opravu faktury.

     - Partnerům se budou fakturovat částky v květnu, pokud jsou nižší než 3. června.
     - Partnerům se fakturuje 3. června vystavených částek, pokud jsou nižší než v květnu.

2. SKUID produktu Project Server 2019 Device CAL se opravil od 0001 do 0003.  

3. Byla odebrána karta Navíc v listu s názvem "List1(2)".


### <a name="next-steps"></a>Další kroky

- Partneři, kteří provádí transakce s časově trvalými softwarovými službami, by si měli stáhnout nejnovější ceník, aby viděli opravené ceny.
