# <a name="june-2021-announcements"></a>Oznámení z června 2021

Tato stránka poskytuje oznámení pro partnerské Centrum Microsoftu pro červen 2021.

________________

## <a name="perpetual-software-price-list-corrected-and-republished"></a><a name="4"></a>Seznam ceníků trvalého softwaru byl opraven a znovu publikován. 

### <a name="categories"></a>Kategorie

- Datum: 2021-06-04
- Nabídky/trhy

### <a name="summary"></a>Souhrn

Po 3. června 2021 byl znovu publikován Ceník trvalého softwaru, který byl vydán od 1. června, aby se opravilo několik problémů. 

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Nepřímá poskytovatelé a partneři s přímým přístupem, kteří v programu Cloud Solution Provider zpracovávají Trvalá softwarová řešení.  

### <a name="details"></a>Podrobnosti

1. Nesprávné ceny v některých měnách

   - Některé produkty v ruštině Rubl (RUB) a Jižní korejštině (KRW) měly nesprávné ceny od 27. dubna 2021 do 3. června 2021. Tyto produkty měly nesprávné ceny, které byly opraveny v aktuálním trvalém ceníku.
 
     - Řada produktů Windows Server
     - Produkty SQL
     - Produkty Office
 
   - Cenové problémy byly zjištěny včas, aby bylo zajištěno, že partneři budou fakturovat správné částky, pokud byly za všechny nákupy přeúčtovány, protože byly zaúčtovány nesprávné ceny. Ovlivnění partneři uvidí na faktuře opravný kredit faktury.

     - Partneři budou Fakturovatelné, pokud jsou nižší než 3 částky z června.
     - Partnerům bude účtováno 3. června, pokud je nižší než květen.

2. Licence CAL zařízení SKUID pro produkt Project Server 2019 byla opravena z 0001 na 0003.  

3. Byla odebrána karta navíc v listu s názvem "List1 (2)".


### <a name="next-steps"></a>Další kroky

- Partneři, kteří používají službu s trvalou podporou softwaru, by si měli stáhnout nejnovější ceník, aby viděli opravené ceny.

## <a name="corrections-to-july-license-based-services-price-list-preview"></a><a name="3"></a>Opravy ceníku pro služby založené na licencích z července Preview

### <a name="categories"></a>Kategorie

- Datum: 2021-06-04
- Nabídky/trhy

### <a name="summary"></a>Souhrn

Neziskové nabídky E1 Office 365 byly v ceníku z července Preview nesprávně označeny jako DEL (odstranění). Aktualizovaný ceník verze Preview byl 4. června 2021 opravit překlepy.

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Nepřímá zprostředkovatelé a poskytovatelé přímých fakturují v programu Cloud Solution Provider služby založené na licencích.

### <a name="details"></a>Podrobnosti

Ceník z července Preview pro služby založené na licencích zveřejněné 1. června 2021 obsahuje překlep pro nabídky Office 365 E1 nezisků. Seznam cen nesprávně označil následující nabídky jako DEL (odstranit). Ceník licenčních služeb byl aktualizován na 4. června 2021 na 11am PST s opravami. Tyto opravy byly platné pouze pro:

- **Název nabídky**: Office 365 E1 (ceny neziskových zaměstnanců)
- **ID nabídky**: 83dca3b9-3d6c-4547-8c5f-ba1f69b130f8 
- **Název nabídky**: Office 365 E1 (ceny neziskových zaměstnanců) – dobrovolná
- **ID nabídky**: f94de68e-F907-471f-b8d3-37ebacbe8ff6 

- **Měny**: vše
- **Před**: nesprávně označeno jako del v ceníku Preview
- **After**: v seznamu cena Preview se zobrazuje jako cesta UNC

Toto byla oprava Preview ceníku, která nemá žádný vliv na fakturaci.

### <a name="next-steps"></a>Další kroky

- Partneři, kteří budou stahovat služby založené na licencích, by si měli stáhnout nejnovější ceník.
