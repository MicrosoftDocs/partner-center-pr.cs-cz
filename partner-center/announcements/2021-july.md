## <a name="readiness-changes-to-the-cloud-solution-provider-csp-customer-address-validation-api-to-go-live-on-july-13-2021"></a><a name="8"></a>Připravenost: Změny rozhraní API pro ověřování Cloud Solution Provider (CSP) zákazníka budou živé 13. července 2021.

### <a name="summary"></a>Souhrn

Změny rozhraní API pro ověření adresy budou živé od 13. července 2021.

### <a name="categories"></a>Kategorie

- Datum: 9. 7. 2021
- Připravenost

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Cloud Solution Provider adresy partnerů s přímým vyúčtováním (CSP) a nepřímých poskytovatelů, kteří vytvářejí nové nebo aktualizují stávající zákazníky

### <a name="details"></a>Podrobnosti

Microsoft běží na vztahu důvěryhodnosti. Zavázali jsme se poskytovat vyhovující, bezpečnou a zabezpečenou metodu ověřování adres zákazníků pro transakce s předplatným zákazníků v programu CSP. Od 13. července 2021 budou změny rozhraní API pro ověření adresy živé.
 
Tyto změny mají vliv pouze na rozhraní API pro ověření adresy. Na rozhraní API pro vytvoření zákazníka a aktualizaci fakturačního profilu to nebude mít vliv. V současné době se navrhovaná adresa nemusí používat s rozhraním API pro vytvoření zákazníka, ale důrazně se doporučuje.

Odpověď vrátí jednu z následujících stavových zpráv:

| Status | Popis | Počet vrácených navrhovaných adres |
|------------|---------------|------------------|
| Ověřená expedice  | Adresa je ověřená a je možné ji poslat na adresu .  | Jednoduché  |
| Ověřené  | Adresa je ověřená.  | Jednoduché  |
| Požadovaná interakce  | Navrhované adresy se výrazně změnily a je potřeba potvrzovat uživatele.  | Jednoduché  |
| Částečná ulice  | Ulice v adrese je částečná a potřebuje další informace.  | Více – maximálně 3  |
| Částečně místní  | Dané prostory (číslo budovy, číslo sady a další) jsou částečné a potřebují další informace.  | Více – maximálně 3  |
| Několik  | Adresa obsahuje několik polí, která částečně zahrnují i část ulice a částečnou část místně.  | Více – maximálně 3  |
| Žádná  | Adresa je nesprávná.  | Žádná  |
| Není ověřeno.  | Adresu nebylo možné odeslat prostřednictvím procesu ověřování.   | Žádná  |

Po odeslané adrese, která se má ověřit prostřednictvím rozhraní API pro ověření adresy, se vrátí následující schéma odpovědi:

```csharp

// <summary>
/// Object represents the address validation response.
/// </summary>

public class AddressValidationResponse
{
   /// <summary>
   /// Gets or sets the original address
   /// </summary>
   /// <value>
   /// Original Address
   /// </value>
   public Address OriginalAddress { get; set; }

   /// <summary>
   /// Gets or sets the suggested addresses
   /// </summary>
   /// <value>
   /// Suggested Addresses
   /// </value>
   public List<Address> SuggestedAddresses { get; set; }

   /// <summary>
   /// Gets or sets the validation status
   /// </summary>
   /// <value>
   /// Status
   /// </value>
   public string Status { get; set; }

   /// <summary>
   /// Gets or sets the validation message
   /// </summary>
   /// <value>
   /// Validation Message
   /// </value>
   public string ValidationMessage { get; set; }

```

Všimněte si, USA v odpovědi vrátí další čtyřmístné přípony pro řádek PSČ, pokud zadáte jenom pět číslic PSČ.

## <a name="keep-your-microsoft-certifications-active-with-help-from-partner-center"></a><a name="6"></a>Aktivní certifikace Microsoftu s pomocí od Partnerské centrum

### <a name="categories"></a>Kategorie

- Datum: 9. 7. 2021
- Rozšiřte své podnikání

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Správci partnerů a globální správci, kteří spravují stav certifikace a prodlužování platnosti od Microsoftu

### <a name="details"></a>Podrobnosti

Zajištění aktivních certifikací Microsoftu je teď snazší díky nápovědě od Partnerské centrum.

Na řídicím panelu služby Partnerské centrum jste vždy mohli zobrazit certifikace vaší organizace, ale teď můžete také zobrazit data vypršení jejich platnosti jak v sestavě dovedností uživatelů, tak v sestavě Partnerské centrum Přehledy Training Report. Tyto aktualizace vám dejte kontrolu nad procesem prodloužení certifikace pro vaši organizaci. Čtěte dál a zjistěte, jak najít aktualizace sestav a jaké budou pro vás přínosné.

V části Kompetence vašeho řídicího Partnerské centrum najdete sestavu kompetence. Při exportu dat do Excel se zobrazí sestava dovedností uživatelů, která obsahuje seznam získaných úspěchů (například certifikací a zkoušek), které vás kvalifikují na kompetence. Na konci této sestavy je nový sloupec s popiskem "Datum vypršení platnosti dovednosti", kde najdete data vypršení platnosti pro certifikace zaměstnanců založené na rolích a speciální certifikace.  

V části Organizační dovednosti řídicího panelu Partnerské centrum najdete sestavu trénování Partnerské centrum Přehledy organizace. Když exportujete data této sestavy do Excel, najdete nový sloupec Expiration Date (Datum vypršení platnosti), který obsahuje data vypršení platnosti pro certifikace vašeho týmu.

Tyto aktualizace vám pomůžou rychle provést akci a připomenou zaměstnancům, aby prodlouili platnost svých certifikací a aby jejich přihlašovací údaje i stav programu MPN byly aktivní.

Jak nový proces prodloužení certifikace funguje?  

- Místo přechodu na Centrum testování a placení na obnovu certifikačních zkoušek budou vaši zaměstnanci navštěvovat Microsoft Learn a předávat bezplatné posouzení obnovy certifikace, které je testuje na nejnovější technologické aktualizace.
- S tím, jak se data vypršení platnosti certifikace blíží, dostanou e-mailová oznámení od Microsoftu a nasměrují je na posouzení obnovení certifikátů na Microsoft Learn.
- Online postupy učení obsahují studijní materiály pro přípravu na posouzení, které se můžou pokusit tolikrát, kolikrát potřebují.
- Předání posouzení prodlouží platnost certifikace po dobu jednoho roku.
- Tento proces se opakuje každý rok, přičemž bude zajištěna aktuálnost dovedností.

V souladu s tímto novým ročním procesem obnovování mění společnost Microsoft také časový rámec platnosti pro role a specializace založené na rolích od dvou let po jeden rok. Od 30. června 2021 budou všechny vytvořené role založené na rolích a specializované certifikace platit po dobu jednoho roku.

Upozorňujeme, že základní a starší verze certifikačních autorit Microsoftu jsou vyloučené a neovlivní nový proces.

### <a name="next-steps"></a>Další kroky

1. přečtěte si datum vypršení platnosti pro certifikace týmu: projděte si zprávu o dovednostech uživatelů nebo školicím centru Přehledy školení a identifikujte datum vypršení platnosti certifikace.
2. Proveďte akci: požádejte tým, aby obnovil své certifikace od Microsoftu před datem vypršení platnosti.

## <a name="dynamics-365-cloud-migration-promotion-and-retirement-of-discounted-offers"></a><a name="1"></a>Dynamics 365 Migrace do cloudu povýšení a vyřazení zlevněných nabídek

### <a name="categories"></a>Kategorie

- Datum: 2021-07-01
- Možnosti

### <a name="summary"></a>Souhrn

Společnost Microsoft se zajímá o spuštění Dynamics 365 Migrace do cloudu propagačních akcí od 1. srpna 2021.

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

partneři Cloud Solution Provider (CSP), Dynamics místních partnerů a lsp

### <a name="details"></a>Podrobnosti

Prostřednictvím programu CSP mohou partneři digitálně transformovat firmy svých zákazníků pomocí špičkových cloudových řešení. v rámci této snahy vyřadíme stávající zlevněné sku (na kartě vyřazené skladové položky v Dynamics 365 Migrace do cloudu propagační a [vyřazené sku Excel souboru](https://partner.microsoft.com/resources/detail/dynamics-365-cloud-promotion-retired-skus-xls)) 31. července 2021 a nahradíte je novým Dynamics 365 Migrace do cloudu povýšení sku.

Společnost Microsoft se zajímá o spuštění Dynamics 365 Migrace do cloudu propagačních akcí od 1. srpna 2021. Toto nové zvýšení úrovně poskytuje zlevněnou slevu Dynamics 365 online služby k tomu, aby zákazníci mohli migrovat migrace do cloudu. Také podporuje zákazníky, kteří zahájili svoji digitální transformaci v rámci starších verzí nabídek migrace za účelem přechodu na nové ceny při povýšení.

#### <a name="dynamics-365-cloud-migration-promotion"></a>Dynamics 365 Migrace do cloudu propagační akce

**Výhody**

Migrace do cloudu propagace nabízí zákazníkům tyto výhody:  

- 40% sleva pro online služby Dynamics 365 s předplatným na tříleté úrovni, která poskytuje předvídatelnost cenové předvídatelnosti během migrace do cloudu.

- Přístupová práva k duálnímu přístupu (souběžné místní i cloudové použití), která po dokončení migrace do cloudu podporuje přístup k uživatelům před místním řešením slunce.

- Podporuje migrace do dvoufázové cloudu a umožňuje organizacím přidávat nové uživatele v období povýšení.

**Podmínky**

Licenční program: CSP

Produkt: Dynamics 365 Migrace do cloudu propagační akce

Propagační sleva: 40%

Doba trvání promoakce: 1. srpna 2021 až 30. června 2023

Termín předplatného: tříleté předplatné; Propagační cena není obnovitelné.

Požadavky na licencování:

- Všechny místní licence Dynamics (Dynamics AX, CRM, partner pro státní správu, NAV, licence k předplatnému a další) s aktivním plánem zákaznických služeb a podpory (například vylepšení plánu nebo výhody) nebo programu Software Assurance (SA) a obnovení do Dynamics 365 online služby.
