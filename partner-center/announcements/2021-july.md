## <a name="software-in-microsoft-china-corporation-limited-mccl-cloud-solution-provider-csp-and-open-license-change"></a><a name="9"></a>Software v programu Microsoft čína Corporation s omezením (MCCL) Cloud Solution Provider (CSP) a Open License změnit

### <a name="categories"></a>Kategorie

- Datum: 2021-07-13
- Možnosti
 
### <a name="summary"></a>Souhrn

po aktualizaci [května 2021](https://partner.microsoft.com/resources/detail/csp-perpetual-software-coming-to-china-pdf) update chceme, abychom vás aktualizovali na náš pokrok s klíčovými oznámeními o softwaru v programu Microsoft čína Corporation s omezením (MCCL) Cloud Solution Provider (CSP) (CSP), které se mění a otevírá program.

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Partneři v programu MCCL CSP

### <a name="details"></a>Podrobnosti

#### <a name="software-in-mccl-csp-general-availability-date"></a>Software v MCCL CSP – datum obecné dostupnosti 

Po první týden ze srpna 2021 budou partneři zaregistrované v programu MCCL CSP prodávat trvalé licence softwaru pro místní nasazení prostřednictvím zprostředkovatele CSP pro MCCL. Zákazníci tak získají flexibilitu licencování, kterou potřebují v rámci své digitální transformace.

#### <a name="open-license-program-retirement"></a>Vyřazení Open License programu

Od 1. ledna 2022 už zákazníci nebudou moct nakupovat nové nebo obnovovat licence na software prostřednictvím programu Microsoft Open Licensee. Do té doby mohou přes program Open License nadále nakupovat všichni zákazníci.

#### <a name="open-license-program-price-change-for-government"></a>Změna ceny Open License programu pro státní správu

Od 1.2021 července se zákazníkům pro státní správu uvidí proměnlivá cena u místních SKU Open License programu, jako je například licence (L), Software Assurance (SA) a licence a Software Assurance společnosti Microsoft (L&SA). Toto je pokračování v práci, která začala v 2018, aby bylo možné zarovnávat cenové nastavení organizací pro státní správu s novým prostředím obchodu. Cena za Open License státní správy se pak bude rovnat nejnižší komerční ceně, přičemž se v programu CSP vytvoří parita s novými cenami za obchodování na úrovni cloudu, která zákazníkům z oblasti státní správy usnadňuje porovnávání řešení. Další informace najdete v tématu [Nejčastější dotazy týkající](https://aka.ms/openlicensegovpricefaq) se cen Open License pro státní správu a [balíčky L100](https://aka.ms/openlicensegovpriceL100).

#### <a name="sa-purchases"></a>Nákupy SA

Výhody SA budou pokračovat až do vypršení platnosti platnosti SA, a to i v případě vypršení platnosti od 1. ledna 2022. Program Microsoft Open Value (OV) se doporučuje pro budoucí nákupy, které zahrnují SA.

>[!NOTE]
>Nejsou k dispozici žádné plány pro změnu OV nebo otevřených předplatných hodnot programu.

## <a name="readiness-changes-to-the-cloud-solution-provider-csp-customer-address-validation-api-to-go-live-on-july-13-2021"></a><a name="8"></a>připravenost: změny v rozhraní API pro ověření adresy zákazníka v Cloud Solution Provider (CSP) pro živý provoz na 13. července 2021

### <a name="summary"></a>Souhrn

Změny v rozhraní API pro ověření adresy budou živé od 13. července 2021.

### <a name="categories"></a>Kategorie

- Datum: 2021-07-09
- Připravenost

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Cloud Solution Provider (CSP) poskytovatelé přímých faktur a nepřímá poskytovatelé, kteří vytvářejí nové nebo aktualizují podrobnosti o adresách stávajících zákazníků

### <a name="details"></a>Podrobnosti

Microsoft běží na důvěryhodnosti. Pro účely transakce zákaznických předplatných v programu CSP jsme zajistili poskytování kompatibilní, bezpečné a zabezpečené metody ověřování adres zákazníků. Od 13. července 2021 budou změny v rozhraní API pro ověření adresy v provozu.
 
Všimněte si, že tyto změny mají vliv pouze na ověřování adres API. Nejsou ovlivněny rozhraní API pro vytváření fakturačních profilů zákazníků a aktualizací. V současné době se navrhovaná adresa nemusí používat s rozhraním API pro vytvoření zákazníka, ale důrazně se doporučuje.

Odpověď vrátí jednu z následujících stavových zpráv:

| Status | Popis | Počet vrácených navrhovaných adres |
|------------|---------------|------------------|
| Ověřené pro přepravce  | Adresa je ověřena a může být expedována.  | Jednoduché  |
| Ověřují  | Adresa je ověřena.  | Jednoduché  |
| Je vyžadována interakce  | Navrhovaná adresa (y) se významně změnila a vyžaduje potvrzení uživatele.  | Jednoduché  |
| Částečně ulice  | Daná ulice v adrese je částečně a potřebuje další informace.  | Víc – maximum ze 3  |
| Částečně místní  | Dané prostory (stavební číslo, číslo sady a další) jsou částečné a vyžadují další informace.  | Víc – maximum ze 3  |
| Několik  | Adresa obsahuje několik polí, která jsou v této adrese částečně (případně i částečně a částečně v ulici).  | Víc – maximum ze 3  |
| Žádná  | Adresa je nesprávná.  | Žádná  |
| Není ověřeno.  | Adresu nelze odeslat prostřednictvím procesu ověřování.   | Žádná  |

Jakmile je adresa odeslaná k ověření prostřednictvím rozhraní API pro ověření adresy, vrátí se následující schéma odpovědi:

```csharp

// <summary>
/// Object represents the address validation response.
/// </summary>

public class AddressValidationResponse
{
   /// <summary>
   /// Gets or sets the original address
   /// </summary>
   /// <value>
   /// Original Address
   /// </value>
   public Address OriginalAddress { get; set; }

   /// <summary>
   /// Gets or sets the suggested addresses
   /// </summary>
   /// <value>
   /// Suggested Addresses
   /// </value>
   public List<Address> SuggestedAddresses { get; set; }

   /// <summary>
   /// Gets or sets the validation status
   /// </summary>
   /// <value>
   /// Status
   /// </value>
   public string Status { get; set; }

   /// <summary>
   /// Gets or sets the validation message
   /// </summary>
   /// <value>
   /// Validation Message
   /// </value>
   public string ValidationMessage { get; set; }

```

Všimněte si, že pro USA odpověď vrátí další příponu se čtyřmi číslicemi pro řádek PSČ, pokud zadáte pouze pět číslic pro poštovní směrovací číslo.
Následuje ukázková odpověď:

```C#

"suggestedAddress": [
{ 
"country": "US", 
"state": "WA", 
"city": "Redmond", 
"addressLine1": "1 Microsoft Way", 
"postalCode": "98052-8300" 
}
], 
"originalAddress": { 
"country": "US", 
"city": "Redmond", 
"addressLine1": "1 Micro Way", 
"postalCode": "98052" 
}, 
"status": "InteractionRequired", 
"validationMessage": "Address field invalid for property: ‘Street’" 
}
```

## <a name="keep-your-microsoft-certifications-active-with-help-from-partner-center"></a><a name="6"></a>Udržování aktivních certifikátů Microsoftu s využitím partnerského centra

### <a name="categories"></a>Kategorie

- Datum: 2021-07-09
- Rozšiřte svou firmu

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Správci partnerů a globální Správci spravující stav a obnovení certifikace Microsoftu

### <a name="details"></a>Podrobnosti

Vaše certifikace Microsoftu je teď teď jednodušší, a to s využitím partnerského centra.

na řídicím panelu partnerského centra jste vždycky viděli certifikace vaší organizace, ale teď můžete také zobrazit jejich datum vypršení platnosti v sestavě dovednosti uživatelů i v sestavě školení Přehledy partnera. Tyto aktualizace zavedou kontrolu nad procesem obnovování certifikace ve vaší organizaci. Přečtěte si, kde najdete informace o tom, jak najít aktualizace sestav a jak vám budou mít výhody.

V části kompetence na řídicím panelu partnerského centra najdete sestavu kompetence. při exportu dat do Excel se zobrazí sestava odbornosti uživatele, která poskytuje seznam získaných výsledků (například certifikace a zkoušky), které vám opravňují k kompetenci. Na konci této sestavy je nový sloupec s názvem "datum vypršení platnosti dovednosti", kde najdete data vypršení platnosti pro vaše zaměstnance a specializace na základě rolí.  

v části dovednosti organizace na řídicím panelu partnerského centra najdete zprávu o školení pro Přehledy partnera. když exportujete data této sestavy do Excel, najdete nový sloupec s datem vypršení platnosti, který obsahuje data vypršení platnosti pro certifikace vašeho týmu.

Tyto aktualizace vám pomůžou rychle provést akci a připomenout si své zaměstnance, aby si obnovili své certifikace a zachovali jejich přihlašovací údaje i stav programu MPN aktivní.

Jak funguje nový proces obnovování certifikátů?  

- Místo přechodu na Centrum testování a placení na obnovu certifikačních zkoušek budou vaši zaměstnanci navštěvovat Microsoft Learn a předávat bezplatné posouzení obnovy certifikace, které je testuje na nejnovější technologické aktualizace.
- S tím, jak se data vypršení platnosti certifikace blíží, dostanou e-mailová oznámení od Microsoftu a nasměrují je na posouzení obnovení certifikátů na Microsoft Learn.
- Online postupy učení obsahují studijní materiály pro přípravu na posouzení, které se můžou pokusit tolikrát, kolikrát potřebují.
- Předání posouzení prodlouží platnost certifikace po dobu jednoho roku.
- Tento proces se opakuje každý rok, přičemž bude zajištěna aktuálnost dovedností.

V souladu s tímto novým ročním procesem obnovování mění společnost Microsoft také časový rámec platnosti pro role a specializace založené na rolích od dvou let po jeden rok. Od 30. června 2021 budou všechny vytvořené role založené na rolích a specializované certifikace platit po dobu jednoho roku.

Upozorňujeme, že základní a starší verze certifikačních autorit Microsoftu jsou vyloučené a neovlivní nový proces.

### <a name="next-steps"></a>Další kroky

1. přečtěte si datum vypršení platnosti pro certifikace týmu: projděte si zprávu o dovednostech uživatelů nebo školicím centru Přehledy školení a identifikujte datum vypršení platnosti certifikace.
2. Proveďte akci: požádejte tým, aby obnovil své certifikace od Microsoftu před datem vypršení platnosti.

## <a name="announcement-of-customer-discount-transparency"></a><a name="4"></a>Oznámení transparentnosti slev zákazníka

### <a name="categories"></a>Kategorie

## <a name="dynamics-365-cloud-migration-promotion-and-retirement-of-discounted-offers"></a><a name="1"></a>Dynamics 365 Migrace do cloudu povýšení a vyřazení zlevněných nabídek

### <a name="categories"></a>Kategorie

- Datum: 2021-07-01
- Možnosti

### <a name="summary"></a>Souhrn

Společnost Microsoft se zajímá o spuštění Dynamics 365 Migrace do cloudu propagačních akcí od 1. srpna 2021.

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

partneři Cloud Solution Provider (CSP), Dynamics místních partnerů a lsp

### <a name="details"></a>Podrobnosti

Prostřednictvím programu CSP mohou partneři digitálně transformovat firmy svých zákazníků pomocí špičkových cloudových řešení. v rámci této snahy vyřadíme stávající zlevněné sku (na kartě vyřazené skladové položky v Dynamics 365 Migrace do cloudu propagační a [vyřazené sku Excel souboru](https://partner.microsoft.com/resources/detail/dynamics-365-cloud-promotion-retired-skus-xls)) 31. července 2021 a nahradíte je novým Dynamics 365 Migrace do cloudu povýšení sku.

Společnost Microsoft se zajímá o spuštění Dynamics 365 Migrace do cloudu propagačních akcí od 1. srpna 2021. Toto nové zvýšení úrovně poskytuje zlevněnou slevu Dynamics 365 online služby k tomu, aby zákazníci mohli migrovat migrace do cloudu. Také podporuje zákazníky, kteří zahájili svoji digitální transformaci v rámci starších verzí nabídek migrace za účelem přechodu na nové ceny při povýšení.

#### <a name="dynamics-365-cloud-migration-promotion"></a>Dynamics 365 Migrace do cloudu propagační akce

**Výhody**

Migrace do cloudu propagace nabízí zákazníkům tyto výhody:  

- 40% sleva pro online služby Dynamics 365 s předplatným na tříleté úrovni, která poskytuje předvídatelnost cenové předvídatelnosti během migrace do cloudu.

- Přístupová práva k duálnímu přístupu (souběžné místní i cloudové použití), která po dokončení migrace do cloudu podporuje přístup k uživatelům před místním řešením slunce.

- Podporuje migrace do dvoufázové cloudu a umožňuje organizacím přidávat nové uživatele v období povýšení.

**Podmínky**

Licenční program: CSP

Produkt: Dynamics 365 Migrace do cloudu propagační akce

Propagační sleva: 40%

Doba trvání promoakce: 1. srpna 2021 až 30. června 2023

Termín předplatného: tříleté předplatné; Propagační cena není obnovitelné.

Požadavky na licencování:

- Všechny místní licence Dynamics (Dynamics AX, CRM, partner pro státní správu, NAV, licence k předplatnému a další) s aktivním plánem zákaznických služeb a podpory (například vylepšení plánu nebo výhody) nebo programu Software Assurance (SA) a obnovení do Dynamics 365 online služby.

- Všechny online služby Dynamics 365 s aktivními předplatnými pro Dynamics 365 ze služby SA nebo ceníku Dynamics 365, doplněk cloudu pro Dynamics, kvalifikovaná nabídka Dynamics 365 nebo nabídka Dynamics 365 Migrace do cloudu.

- Tato propagační akce není platná pro zákazníky s čistým novým cloudem ani pro neaktivní zákazníky v místním prostředí. (Další informace najdete v tématu [zásady online služby](https://www.microsoft.com/licensing/terms/productoffering/MicrosoftDynamics365Services/EAEAS).)

**Skladové položky**

úplný seznam sku najdete v tématu karta povýšení sku v Dynamics 365 Migrace do cloudu propagační a [vyřazené sku Excel souboru](https://partner.microsoft.com/resources/detail/dynamics-365-cloud-promotion-retired-skus-xls)).
