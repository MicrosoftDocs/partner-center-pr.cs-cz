## <a name="csp-perpetual-software-coming-soon-to-china"></a><a name="5"></a>Software CSP trvalá už brzy k Číně

### <a name="categories"></a>Kategorie

- Datum: 2021-05-05
- Možnosti

### <a name="summary"></a>Souhrn

Při cílení na první čtvrtletí fiskálního roku (FY22 Q1) bude k dispozici Trvalá softwarová aplikace Microsoft Čína Corporation s omezením (MCCL) CSP.

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Partneři v programu MCCL CSP

### <a name="details"></a>Podrobnosti

Program CSP byl exponenciálně vzrostl s více než 90 000 zúčastněnými společnostmi. Jedná se o náš primární partnerský prodejní pohyb pro malé a midsized firmy i pro menší podnikové zákazníky v rámci nového prostředí pro obchod. Partneři můžou zákazníkům vytvářet a dodávat odlišné nabídky, prodávat řešení Microsoft Cloud portfolio a řešení třetích stran od komerčního tržiště Microsoftu spolu s vlastními řešeními a službami přidanými s přidanou hodnotou. Můžou spravovat celý životní cyklus Customer-Engagement a zajistit jejich důvěryhodnost zákazníkům jako důvěryhodné poradce.

#### <a name="expanding-opportunities-for-partners"></a>Rozšiřování příležitostí pro partnery

V tomto potenciál Microsoft rozšiřuje příležitosti pro partnery CSP s vydáním komerčních trvalé licence softwaru bez programu Software Assurance (SA) pro místní nasazení prostřednictvím služby * MCCL CSP, která cílí na Q1 v Microsoft FY22. Tato možnost zákazníkům poskytuje flexibilitu licencování, kterou potřebují v rámci své digitální transformace.

Jak [už bylo oznámeno](https://blogs.partner.microsoft.com/mpn/expanding-opportunities-for-partners-in-the-cloud-solution-provider-program/)od 1. ledna 2022, nemůžete prostřednictvím Open License programu provádět žádné obnovení SA ani nové nákupy softwarových licencí. Nové nákupy jenom pro licence by se měly provádět pomocí MCCL CSP.

* To platí jenom pro MCCL CSP. Zprostředkovatel CSP společnosti 21Vianet nemá žádný vliv.

### <a name="next-steps"></a>Další kroky

Partneři, kteří se zaregistrovali nebo brzy zaregistrovali v MCCL CSP, by se měly připravit kontrolou následujících materiálů a sdílení s příslušnými zúčastněnými stranami v rámci organizace:

- [Materiály k poskytovateli CSP a partnerského centra pro větší čínskou oblast (GCR)](https://partner.microsoft.com/resources/collection/software-in-csp-cht#/): Úvodní přehled, pokyny k licencování, řazení partnerského centra a postupy: videa, nejčastější dotazy a další informace o rozhraní API. Titulky jsou k dispozici v zjednodušené čínštině nebo tradiční čínštině.

- Informace [o programu microsoft Open License program ve zprávách o licencování společnosti Microsoft](https://www.microsoft.com/zh-cn/licensing/news/microsoft_open_license_program_changes): Další podrobnosti týkající se změn Open License programu a možností licencování. Přečtěte si také související Open License program mění informace o připravenosti na [webu partnera Microsoftu](https://partner.microsoft.com/resources/collection/csp-open-evolution-to-a-better-experience#/).

## <a name="final-reminder-deprecation-of-get-qualification-on-may-6-2021"></a><a name="4"></a>Konečné připomenutí: vyřazení získání kvalifikace z května 6, 2021

### <a name="categories"></a>Kategorie

- Datum: 2021-05-04

- Možnosti

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Partneři, kteří prodávají akademické a neziskové a státní komunitní Cloud (RSZ) prostřednictvím programu Cloud Solution Provider pomocí rozhraní API partnerského centra

### <a name="details"></a>Podrobnosti

Toto oznámení je následná vylepšení partnerského centra [vydané v prosinci](https://docs.microsoft.com/partner-center/announcements/2020-december#1). V rámci této verze se nasadila nová rozhraní API GET a POST a v důsledku toho se **stávající kvalifikace Get vyřadí do 6. května 2021**. V tuto chvíli se budete muset převést na použití nového rozhraní API pro POST Center. Nová rozhraní API pro odesílání vám umožní koupit nabídky vzdělávání, zatímco nové rozhraní API pro získání vám umožní koupit předem kvalifikované nabídky a nabídky RSZ.

### <a name="next-steps"></a>Další kroky

- **Aktualizujte nové rozhraní API** pro úspěšný a včasný přechod.

- **Projděte si nové změny a příručka k rozhraní API partnerského centra** v prostředcích připravenosti operací: [vylepšení procesu ověřování zákazníků z partnerského centra](https://partner.microsoft.com/resources/collection/partner-center-edu-validation-enhancements#/).

- Tyto informace můžete sdílet s příslušnými týmy ve vaší organizaci a prodejci, kteří jim pomůžou tyto změny připravit.

Pokud máte dotazy související s tímto oznámením, obraťte se na [podporu partnerského centra](https://partner.microsoft.com/dashboard/support/referrals/servicerequests?category=referrals).

## <a name="corrections-to-may-1-2021-license-based-services-price-list"></a><a name="3"></a>Opravy pro ceník 1. května 2021 služby na základě licencí

### <a name="categories"></a>Kategorie

- Datum: 2021-05-04

- Nabídky/trhy

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Nepřímá zprostředkovatelé a poskytovatelé přímých fakturují v programu Cloud Solution Provider služby založené na licencích.

### <a name="details"></a>Podrobnosti

Ceník pro služby založené na licencích zveřejněné 1. května 2021 zahrnoval nesprávnou cenu za USD pro Microsoft 365 Business Standard. Publikovaný Ceník byl $8, pokud by měl být $10. Ceník služeb založených na licencích se aktualizoval 3. března 2021 na 10:30 PST s opravami. Tyto opravy byly platné pouze pro:

- Název nabídky: Microsoft 365 Business Standard

- ID nabídky: 031c9e47-4802-4248-838e-778fb1d2cc05

- Měny: USD

- Před: list Price: $8

- Za: list Price: $10

Tyto změny se vztahují pouze na výše uvedenou nabídku. Jiné nabídky neobsahovaly žádné opravy. Toto byla pouze chyba ceníku, nikoli skutečná chyba ceny, která bude během této doby mít vliv na fakturaci.

## <a name="announcement-of-state-owned-entity-qualification"></a><a name="2"></a>Oznámení kvalifikace vlastněné státem

- Datum: 4. 5. 2021
- Možnosti

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

### <a name="details"></a>Podrobnosti
