## <a name="hosting-and-spla-reseller-incentive-programs-extended-through-september-30-2021"></a><a name="8"></a>Hostování a SPLA prodejců pro prodejce, které jsou prodloužené do 30. září 2021

### <a name="categories"></a>Kategorie

- Datum: 2021-05-07
- Možnosti

### <a name="summary"></a>Souhrn

Přidaný program pro maloobchodního přidaných hostitelských služeb a licenční smlouvy s poskytovatelem služeb (SPLA) pro FY21 se rozšířil do 30. září 2021. FY22 programy začnou od 1. října 2021.

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Účastníci hostování programu pro pobídku a SPLA prodejce

### <a name="details"></a>Podrobnosti

FY21 hostující program a motiv FY21 SPLA pro prodejce, který původně končí 30. června 2021, se v průběhu 30. září 2021 prodlouží. Byly aktualizovány příručky programu, aby odrážely změnu. Aktuální tarify budou pokračovat do 30. září 2021.

FY22 hostující program a motiv FY22 SPLA pro prodejce začnou být od 1. října 2021. Další informace o těchto programech budou sdíleny v srpnu 2021.

### <a name="next-steps"></a>Další kroky

Sdílejte prosím tyto informace s vaší organizací, aby bylo možné maximalizovat své možnosti. Partneři si můžou prohlédnout [nejnovější Průvodce pobídkou a podpůrné prostředky](https://partner.microsoft.com/membership/partner-incentives).

## <a name="default-bank-profile-functionality-is-available-in-partner-center-for-incentive-enrollment"></a><a name="7"></a>V partnerském centru jsou k dispozici výchozí funkce profilu banky pro motivační zápis.

### <a name="categories"></a>Kategorie

- Datum: 2021-05-07
- Možnosti

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Správci pobídek

### <a name="details"></a>Podrobnosti

S radostí oznamujeme dostupnost výchozí funkce bankovního profilu v Partnerské centrum. Díky této funkci můžou správci pobídek nastavit výchozí bankovní profily a přiřazovat je Microsoft Partner Network umístěním (MPN). Tyto výchozí profily po nastavení bude následně používat Microsoft, když se partner chce zaregistrovat do jakéhokoli programu pobídek pro toto umístění MPN.

Podrobné pokyny najdete v tématu [Nastavení výchozího bankovního profilu](https://docs.microsoft.com/partner-center/incentives-create-and-manage-your-payout-and-tax-profiles#set-up-a-default-bank-profile).

#### <a name="enrolling-in-an-incentive-program-when-a-default-bank-profile-has-been-assigned"></a>Registrace do programu pobídek při přiřazení výchozího bankovního profilu

Pokud se pokoušíte zaregistrovat umístění MPN v programu pobídek a toto umístění už má nastavený výchozí profil, zobrazí se vám výchozí profil, když vyberete Zaregistrovat a přijmete pozvánku. Zobrazí se také jakýkoli dostupný daňový profil, který jste vytvořili pro toto umístění MPN. Pokud má Microsoft všechny požadované podrobnosti o bankovním a daňovém profilu, zobrazí se výzva k dokončení registrace výběrem možnosti Odeslat.

Máte také možnost zvolit bankovní profil, který se liší od výchozího bankovního profilu. Pokud Microsoft vyžaduje další podrobnosti o platebním nebo daňovém profilu nebo měně, budete vyzváni k výběru možnosti Pokračovat a přesměrování na stránku Platební a daňový profil, na které se zobrazí chybějící podrobnosti.

Registrace se považuje za dokončenou, pouze pokud Microsoft ověří platební a daňový profil, který jste pro registraci poskytli. Další informace o platebním a daňovém profilu [najdete tady.](https://docs.microsoft.com/partner-center/incentives-create-and-manage-your-payout-and-tax-profiles)

### <a name="questions"></a>Máte otázky?

## <a name="csp-perpetual-software-coming-soon-to-china"></a><a name="5"></a>Časově neomezené software CSP již brzy bude v Číně

### <a name="categories"></a>Kategorie

- Datum: 5. 5. 2021
- Možnosti

### <a name="summary"></a>Souhrn

Cílení na první čtvrtletí fiskálního roku (FY22 Q1) bude časově omezený software k dispozici programu CSP Microsoft China Corporation Limited (MCCL).

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Partneři v programu MCCL CSP

### <a name="details"></a>Podrobnosti

Program CSP se exponenciálně rozrostl s více než 90 000 zapojených společností. Jedná se o hlavní prodejní pohyb partnerů pro malé a střední firmy i pro menší firemní zákazníky v rámci nového komerčního prostředí. Partneři můžou zákazníkům vytvářet a dodávat odlišné nabídky, prodávat řešení Microsoft Cloud portfolio a řešení třetích stran od komerčního tržiště Microsoftu spolu s vlastními řešeními a službami přidanými s přidanou hodnotou. Můžou spravovat celý životní cyklus Customer-Engagement a zajistit jejich důvěryhodnost zákazníkům jako důvěryhodné poradce.

#### <a name="expanding-opportunities-for-partners"></a>Rozšiřování příležitostí pro partnery

V tomto potenciál Microsoft rozšiřuje příležitosti pro partnery CSP s vydáním komerčních trvalé licence softwaru bez programu Software Assurance (SA) pro místní nasazení prostřednictvím služby * MCCL CSP, která cílí na Q1 v Microsoft FY22. Tato možnost zákazníkům poskytuje flexibilitu licencování, kterou potřebují v rámci své digitální transformace.

Jak [už bylo oznámeno](https://blogs.partner.microsoft.com/mpn/expanding-opportunities-for-partners-in-the-cloud-solution-provider-program/)od 1. ledna 2022, nemůžete prostřednictvím Open License programu provádět žádné obnovení SA ani nové nákupy softwarových licencí. Nové nákupy jenom pro licence by se měly provádět pomocí MCCL CSP.

* To platí jenom pro MCCL CSP. Zprostředkovatel CSP společnosti 21Vianet nemá žádný vliv.

### <a name="next-steps"></a>Další kroky

Partneři, kteří se zaregistrovali nebo brzy zaregistrovali v MCCL CSP, by se měly připravit kontrolou následujících materiálů a sdílení s příslušnými zúčastněnými stranami v rámci organizace:

- [Materiály k poskytovateli CSP a partnerského centra pro větší čínskou oblast (GCR)](https://partner.microsoft.com/resources/collection/software-in-csp-cht#/): Úvodní přehled, pokyny k licencování, řazení partnerského centra a postupy: videa, nejčastější dotazy a další informace o rozhraní API. Titulky jsou k dispozici v zjednodušené čínštině nebo tradiční čínštině.

- Informace [o programu microsoft Open License program ve zprávách o licencování společnosti Microsoft](https://www.microsoft.com/zh-cn/licensing/news/microsoft_open_license_program_changes): Další podrobnosti týkající se změn Open License programu a možností licencování. Přečtěte si také související Open License program mění informace o připravenosti na [webu partnera Microsoftu](https://partner.microsoft.com/resources/collection/csp-open-evolution-to-a-better-experience#/).
