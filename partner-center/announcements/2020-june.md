## <a name="now-available-windows-7-year-2-esu-for-cloud-solution-provider-csp"></a><a name="6"></a>Nyní dostupná Windows 7. rok 2 EVJ pro Cloud Solution Provider (CSP)

### <a name="categories"></a>Kategorie

- Datum: 2020-06-29
- Typ: nabídky/trhy
- Stav: nyní k dispozici

### <a name="summary"></a>Souhrn

Ceny za rozšířené aktualizace zabezpečení v systému Windows 7, 2 a roky jsou nyní v [galerii prostředků](https://partner.microsoft.com/resources/collection/windows-esu-csp#/).

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Všichni partneři účastnící se programu Cloud Solution Provider (CSP)

### <a name="details"></a>Podrobnosti

Podpora pro Windows 7 skončila 14. ledna 2020. I když mnoho zákazníků je dobře na svém způsobu, jak nasazovat Windows 10, znamená to, že každý z nich je v rámci procesu upgradu v jiném bodě. Doba platnosti Windows 7 rok 1 EVJ končí 12. ledna 2021. Oznamujeme, že zákazníci začínají na příští rok.  

Pro zákazníky, kteří očekávají, že budou používat zařízení s Windows 7 v 2021, vydáváme zákazníkům k dispozici ceny pro Windows 7 v roce 2 EVJ (Cloud Solution Provider) nyní, aby mohli zákazníkům pomáhat s rozhodnutím Windows 10. Další informace najdete [v galerii prostředků](https://partner.microsoft.com/resources/collection/windows-esu-csp#/) .

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>Aktualizace funkcí pro odebrání vztahu partnerského centra

### <a name="categories"></a>Kategorie

- Datum: 2020-06-29
- Typ: možnosti
- Stav: nyní k dispozici
- 
### <a name="summary"></a>Souhrn

Od 29. června 2020 nepřímá poskytovatelé Cloud Solution Provider (CSP) a nepřímo prodejci můžou odebrat podřízené relace pomocí partnerského centra a nebude už muset odeslat lístek podpory pro partnerský partner. Tím se při provádění těchto aktivit uloží značná doba pro partnery.

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Nepřímá poskytovatelé a nepřímo prodejci Cloud Solution Provider (CSP)

### <a name="scenarios-supported"></a>Podporované scénáře:

- Nepřímá poskytovatelé, kteří mají vytvořenou relaci s nepřímým prodejcem, mohou tuto relaci odebrat pomocí partnerského centra.
- Nepřímý prodejci, kteří mají vytvořenou relaci s koncovým zákazníkem, můžou tuto relaci odebrat pomocí partnerského centra.

Akce ve výše uvedených scénářích musí provádět agent pro správce partnerského centra.

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>Výchozí četnost fakturace se mění z roční na měsíčně.

### <a name="categories"></a>Kategorie

- Datum: 2020-06-17
- Typ: možnosti
- Stav: již brzy

### <a name="summary"></a>Souhrn

Četnost fakturace ve výchozím nastavení se v rámci nákupu v partnerském centru mění z roční na měsíční.

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Poskytovatelé přímého poskytovatele Cloud Solution Provider (CSP) a nepřímá zprostředkovatelé

### <a name="details"></a>Podrobnosti

Nastavení četnosti fakturace určuje, jak často se má partner účtovat za služby, které zakoupili. Partner uvidí toto nastavení v kroku **recenze** při nákupu v katalogu partner Center. Toto nastavení má po dobu od základu pro licenční služby nastavené na **rok roční** . Na základě zpětné **vazby od partnera** tým partnerského centra mění toto výchozí nastavení od 17. června Po **měsíc** , takže lépe se bude zarovnávat s tím, co partneři očekávají. Partneři je mohou změnit před odesláním transakce na stránce pro kontrolu nebo po transakci ze stránky Správa předplatného. Je potřeba si uvědomit, že nabídky, které podporují jenom roční fakturaci, budou dál platit jako výchozí. Tato výchozí změna platí jenom pro uživatelské prostředí portálu partnerského centra, ale nemění se způsob fungování rozhraní API partnerského centra.

Doufá žeme, že tato změna snižuje možný potenciál pro neúmyslné nákupu ročních předplatných.

## <a name="changes-to-microsoft-365-f1"></a><a name="3"></a>Změny Microsoft 365 F1

### <a name="categories"></a>Kategorie

- Datum: 2020-06-15
- Typ: nabídky/trhy
- Stav: nyní k dispozici

### <a name="summary"></a>Souhrn

Microsoft 365 F1 se budou aktualizovat novými službami. Do 15. června se uvolní nová nabídka, která nahradí předchozí F1.

### <a name="impacted-audience"></a>Ovlivněná cílová skupina

Poskytovatelé přímého poskytovatele Cloud Solution Provider (CSP) a nepřímá zprostředkovatelé

### <a name="details"></a>Podrobnosti

Od dubna 2020 byla v partnerském centru k dispozici Microsoft 365 F1. Vylepšení se přidávají přidáním podpory pro nové služby: možnosti služby Exchange pro terminál a službu MS Search. Nové služby se zřídí automaticky na všech stávajících předplatných založených na původních nabídkách F1 na konci června. Původní nabídky F1 budou nahrazeny 15. června s novými nabídkami F1 (níže) pro nové nákupy předplatného.

Původní nabídky F1:

   |**Název nabídky**|**ID nabídky**|**ID materiálu**|**ID zřizování**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|1ce4a4ba-65f7-49e7-bb28-d92d07d58b86|1PI-00005|M365_F1|
   |Zkušební verze Microsoft 365 F1|dd97e36a-f0df-4a9a-b461-625420656511|1PI-00006|M365_F1|

Partneři, kteří chtějí koupit nové odběry F1, by měli použít následující ID nabídky k 15. června. Partneři uvidí tato ID v katalogu partnerského centra. Partneři, kteří využívají rozhraní API partnerského centra, můžou použít i tyto informace.

Nové nabídky F1:

   |**Název nabídky**|**ID nabídky**|**ID materiálu**|**ID zřizování**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|17b4a1d6-2c65-478e-a046-51cf40d25d76|1PI-00012|M365_F1_COM|
   |Zkušební verze Microsoft 365 F1|2e4e9424-06db-4d35-b2d8-3a1b0ef1dc63|1PI-00013|M365_F1_COMM|

### <a name="call-to-action"></a>Zavolat na akci
Partneři s aktuálním Microsoft 365m produktem F1 nemají žádnou akci. Plány služeb pro veřejné terminály a služby MS Search budou na konci června na svých stávajících plánech služeb zpětně vyplněny.

Partneři, kteří chtějí koupit Microsoft 365 F1 po 15. června, by měli použít nové informace, které vám pomohou zajistit, aby si koupili správnou nabídku. Nový kód nabídky F1 a další informace budou aktualizovány v červenci aktualizace ceníku Cloud Solution Provider (CSP) od 1. července.
