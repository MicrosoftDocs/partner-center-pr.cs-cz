# <a name="how-csp-program-partners-can-sell-subscriptions-to-customers"></a>Jak mohou partneři programu CSP prodávat předplatná zákazníkům

**Odpovídající role:** Agent pro správu | Globální správce | Agent helpdesku | Agent prodeje

Jako partner CSP můžete svým zákazníkům nabízet předplatná produktů Microsoftu. V Partnerské centrum můžete spravovat všechna předplatná pro každého zákazníka. Můžete přidat předplatná, přidat licence ke stávajícím předplatným a dokonce nabídnout zkušební předplatná zákazníkům, kteří chtějí software otestovat. Kromě toho můžete zobrazit analýzy, abyste porozuměli tomu, jak vaši zákazníci software používají, což vám pomůže poskytovat lepší podporu.

|**Chcete-li to provést**   |**Přečtěte si:**   |
|----------------------|:----------------------|
|Vytváření a správa předplatných pro stávající zákazníky v Partnerské centrum.|[Vytvoření nového předplatného](create-a-new-subscription.md)|
|Po přidání předplatného můžete zaměstnancům zákazníka přiřadit licence.  |[Přiřazení licencí k uživatelům](assign-licenses-to-users.md)|
|Můžete také spravovat hromadné aktualizace licencí.   |[Přiřazení licencí několika uživatelům nebo jejich odvolání](bulk-license-provisioning-for-multiple-users.md)|
|Když zákazník přidá nové zaměstnance, můžete do stávajících předplatných přidat další licence.   |[Přidání licencí nebo služeb do stávajícího předplatného](add-licenses-or-services-to-an-existing-subscription.md)|
|Zákazník může chtít software vyzkoušet ještě před tím, než se do něj zahodí, takže nabídne zkušební předplatná.    |[Nabídka zkušebních verzí produktů Microsoftu zákazníkům](offer-your-customers-trials-of-microsoft-products.md)|
